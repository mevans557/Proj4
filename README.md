This is a collection of code snippets and Latex files for the creation of my 4th year Project, as part of my Masters MMath degree. The C++ code is deprecated, but is included for posterity.

The python files contain both new and old code, the code used to generate the images in the report is located in the files beginning SIS, and the griddify.py file was written to allow better calculation of means and standard deviations.

The poster and report folders contain latex files, bib files, and figures used to create the report and poster for submission. These use the templates provided by Durham University.

The commit history may be interesting to those wishing to see the evolution of this project over time.
